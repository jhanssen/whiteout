# ts-blank-space fixture corpus

The files in `cases/` and `expected/` are copied verbatim from
[ts-blank-space](https://github.com/bloomberg/ts-blank-space)'s
`tests/fixture/cases/` and `tests/fixture/output/` directories. They are
included here only as test inputs and expected outputs; the corpus harness
(`tests/ts_blank_space_corpus.c`) runs each fixture through whiteout and
classifies the result.
